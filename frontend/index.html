Under The Hood

How It Works

A 4-stage RAG pipeline that combines OCR, semantic search, and LLM reasoning to deliver actionable compliance insights.

1

OCR Extraction

Extracts text from PDFs and images using PyMuPDF + Tesseract with table-aware preprocessing

2

Smart Chunking

Splits documents into semantically meaningful sections based on insurance-specific headings

3

RAG Retrieval

Finds relevant Indian insurance laws using FAISS vector search + sentence transformer embeddings

4

LLM Reasoning

Groq-hosted Llama 3.1 assesses compliance with strict JSON output and hallucination filtering

Why This Matters

Key Features

Unfair Clause Detection

Flags problematic clauses with severity ratings (High / Medium / Low) and explains exactly why each one is harmful to the policyholder.

Hallucination Filtering

Every flagged clause is validated against the original document. If the LLM fabricated it, it gets filtered out automatically.

Real Legal Knowledge Base

Cross-references against 400+ indexed clauses from the Motor Vehicles Act 1988, IRDAI Regulations, and the Motor Insurance Handbook.

Structured Confidence Scoring

AI-generated confidence scores with separate assessments for pricing fairness, legal adherence, and coverage adequacy.
